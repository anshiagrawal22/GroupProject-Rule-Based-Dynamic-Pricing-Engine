# Rule-Based Dynamic Pricing Engine

A web-based pricing engine that dynamically adjusts prices using predefined business rules and fairness constraints to ensure transparent and balanced pricing decisions.

## Features
- Dynamic price calculation for Hotel booking based on season, check-in day, and demand level  
- Transparent step-by-step pricing breakdown  
- Fairness constraints with price cap and floor  
- Manual and auto-predict pricing modes  

## Tech Stack
- Python  
- Flask  
- HTML/CSS  
- JavaScript
